# История изменений

## v1.0.0 — 2026-07-22

Первый публичный выпуск. Клиент протокола LACERT на C для ESP-IDF — прошивка
для устройств ESP32, совместимая по проводу со шлюзом байт-в-байт.

### Что внутри
- Рукопожатие, передача данных, ротация ключа, ответ на проверку целостности
  прошивки; крипто через mbedTLS (ECDSA, ChaCha20, SHA-256) + ML-KEM и BLAKE3.
- Один проект собирается под XIAO ESP32-C6 (RISC-V), ESP32-S3 и
  ESP32-S3-N8R2 (Xtensa).
- Аппаратный источник случайности ESP32 (`esp_fill_random`) для ML-KEM.

### Сборка
Сторонние компоненты (ML-KEM из PQClean, BLAKE3) не хранятся в репозитории —
скачиваются `scripts/fetch-components.sh` на зафиксированных версиях, поверх
накладывается локальная обвязка из `component-overlay/`. Затем `idf.py build`.
Подробно — в `docs/*/FIRMWARE_BUILD.md`. Сборка под esp32c6 проверена из
чистого клона.
