# История изменений

## v1.0.2 — 2026-07-23

Уточнение инструкций по сборке.

### Изменено
- В `FIRMWARE_BUILD.md` и `LINUX_DEBUG.md` добавлено пояснение к путям. Команды
  в документации записаны для монорепозитория, где прошивка лежит в подкаталоге
  `firmware/`, а в этом репозитории она находится в корне — шаг `cd firmware`
  здесь пропускается.

## v1.0.1 — 2026-07-22

Выпуск дополняет репозиторий прошивки отладочными материалами и уточняет
документацию.

### Добавлено
- Англоязычная методика замеров на платах (`BOARD_MEASUREMENTS.md`).

### Изменено
- `NOTICE` дополнен: отладочный клиент `linux-debug/` линкуется с OpenSSL
  (системная зависимость, в репозитории не поставляется).
- Год в лицензии и уведомлении обновлён.

## v1.0.0 — 2026-07-22

Первый публичный выпуск. Клиент протокола LACERT на C для ESP-IDF: прошивка для
ESP32, совместимая со шлюзом по проводу. Рукопожатие, передача данных, ротация
ключа, ответ на проверку целостности прошивки.
